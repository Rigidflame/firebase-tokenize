firebase-tokenize
==============

Easily generate Firebase tokens from the command-line!

Installation
------------

Install via npm:

    npm install -g firebase-tokenize


Example
-------
  firebase-tokenize -n yourFirebase '{"uid": "america", "is": "bestest"}'
